Blog Site Using Django

Using python's "Django" framework, I made a prototype for an online blog site where users can sign up using their email address, then make posts and share images. Users can also tag their posts and search for posts via post title, or using the posts' tags. Making this project taught me a lot about backend web design, databases, web hosting and deployment. This project was made with help from online tutorials, however many of the features were added by myself.

Simple Inventory Sites Using Django and Flask

I built two simple CRUD apps using both the Django and Flask web frameworks. These apps allow users to create, read, update and delete items in a web database. The Django version also has an API that was built using the Django REST API framework, and allows users to obtain JSON data about the items in the database directly via the API. This project was a good introduction to backend web frameworks and taught me a lot about databases and APIs.

Audio Player for Looping Audio Tracks

Using pySimpleGUI, pygame, pydub, and a number of other packages, I created a music player with an interface that allows the user to easily create custom loops of songs. The start and end point of these loops can be easily changed and saved for future use. Custom loops can also be exported as .mp3 files for playback in other environments. This was one of my earlier projects, and although it took a long time, I'm pleased that I could implement all of the features that I wanted for this app.

Simple Fighting Game Using HTML and JS

This was the first project that I made after I started learning to code. It is a simple 2 player fighting game, where the players can choose a character and fight against each other in a best of three match. This project taught me a lot about Javascript and object oriented programming. Although the game contains most of the basic features of a fighting game, I would like to add more moves, stages, characters and music at some point in the future.
